废土信息学修改存档实现自定义

01. 学科介绍

不少玩家都希望能在游戏中自定义角色和卡车, 可惜游戏并未提供这些功能(自定义初始角色在 DLC 死者国度已经实现), 你当然可以通过制作 Mod 来解决这种需求, 但这种做法的学习成本就有点太高了, 有没有简单一些的途径呢, 当然有, 那就是直接修改存档, 值得一提的是, 即使未来游戏提供了更多关于自定义的功能, 修改存档也还是有价值的, 例如你想要某个特定的特质, 但又不想 SL 浪费时间.

游戏的存档一般被保存在这个路径: C:\Users\用户名\AppData\LocalLow\NeutronStar\WastelandExpress, 如果实在找不到, 那么在 C 盘搜索 NeutronStar 也行(该单词的中文含义为"中子星", 既游戏开发工作室的名称), 就是会慢点(几十秒甚至几分钟), 在目标文件夹中, 你能发现一些 JSON 文件, 这就是存档了, JSON 是种轻型的数据存储模式, 文本都是未经压缩的明文, 修改起来也十分方便.

注意: 修改存档之前, 先复制一份存档到别的位置进行备份, 以免修改失败导致烂档, 推荐使用 Notepad++ 等文本编辑工具打开这些 JSON 文件, 因为有代码高亮, 方便进行查看和编辑, 但如果实在没有或懒得安装, 用系统自带的 TXT 文档编辑器打开也行, 现在的存档经过压缩变得较为难看了, 可用 在线 JSON 格式化工具 进行"格式化校验", 修改后无需重新压缩, 直接保存修改, 进入游戏读取即可.

02. 修改角色

首先你需要定位到角色属性在存档中的位置, 以角色的姓名进行搜索定位即可(存档过去会记录角色的职业, 但改版后现在就只会记录角色的姓名), 我们在搜索结果的位置会看到以下的内容, 这是该角色的各项属性和注释, 修改数值之后, 保存文件再读取存档即可, 下面的 // 和后面的中文是注释, 注意别把斜杠和中文也写进文件, 因为 JSON 文件不支持注释, 这可能导致错误, 总之只改数值就行了.

修改完角色能力后, 稍微把文档往下滚动, "PeopleBuff" 就是角色的特质代码了, 既"同性恋"和"惹是生非"等特性, 如果不想要某个特质, 将其删除即可, 如果想添加特质, 可通过查找其他角色身上的特质, 把那部分代码复制粘贴到 "PeopleBuff" 后面的括号里就行(注意, 添加时在 } 后面得用 , 进行分割), 其他角色的初始特质可参考 角色列表.html, 特质代码和详情可参考 人物特质.html.

修改装备也是一样的道理, 在特质上方的 "Equip" 属性就是装备了, 可以在做出一件传奇装备后, 通过复制粘贴的方法给所有角色都装备上, 但这就有点耍无赖了, 有兴趣就自行研究吧, 这里就不展示了, 修改后记得保存文件, 然后在游戏中读取即可, 你不需要退出游戏或重启, 文件保存后在游戏中读取存档, 在载入游戏的界面, 存档会显示文件的最后的修改时间, 直接加载就能知道是否修改成功了.

03. 定义卡车

可通过搜索 _autoMain 关键词定位到卡车的数据, XJ 只修改过马力, 也就是 "EnginePower": 8, 和 "Power": 8.0,, 因为之前的版本有 BUG, XJ 等不及开发团队修复, 就自己动手了, 总而言之, 不懂的数据就自己尝试, 存档有备份就行, 注意不要一次性改太多东西, 否则出错了你也不知道是哪里的改动不对, 那么就先说这么多吧, 驾驶不规范, 丧尸乐开花, 废土狂飙有风险, 请注意交通安全.
